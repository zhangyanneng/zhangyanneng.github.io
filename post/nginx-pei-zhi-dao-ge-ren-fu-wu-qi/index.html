Nginx配置到个人服务器

2019-06-19

1.安装

apt-get install software-properties-common add-apt-repository ppa:nginx/stable apt-get update apt-get install nginx

2.安装目录查找方式

whereis nginx

nginx: /usr/sbin/nginx /etc/nginx /usr/lib/nginx /usr/share/nginx /usr/share/man/man8/nginx.8.gz

对应的几个目录就是 /usr/sbin/nginx /etc/nginx /usr/lib/nginx /usr/share/nginx /usr/share/man/man8/nginx.8.gz

3.启动

/usr/sbin/nginx

4.检查配置状态

nginx -t 提示如下：

nginx: the configuration file /etc/nginx/nginx.conf syntax is ok
nginx: configuration file /etc/nginx/nginx.conf test is successful

5.停止

nginx -s stop

6.重启

service nginx restart

6.卸载

apt-get remove nginx # 删除nginx，保留配置文件 rm -rf /etc/nginx #删除配置文件
